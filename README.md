# Component

## React is all about components

Creating components using functions, should use capital letter

```JSX

function App() {
 return (
   <div className="App">
     <h1>Hello World</h1>
   </div>
 );
}

```

```JSX
const App = () => {
 return (
   <div className="App">
     <h1>Hello World</h1>
   </div>
 );
}
```

```JSX
const App = () => (
   <div className="App">
     <h1>Hello World</h1>
   </div>
 );
```
